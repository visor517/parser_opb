Парсер для сбора тестов по основам промышленной безопасности.
Собирает тесты с https://tests24.su/a1-osnovy-promyshlennoj-bezopasnosti-2019/
Написан на Phyton

Описание:
1. Подключаются библиотеки:
	selenium - для открытия в браузере решенного теста
	beautifulsoup4 - для сбора информации
2. Подключается Cromedriver
3. Создается файл для теста
4. Перебираются 16 страниц с билетами
4.1. На странице билета открываются все скрытые вопросы, чтобы можно было ответить на все сразу без переходов. Добавлением скрипта на JS
4.2. В каждом вопросе selenium прожимает первый вариант
4.3. Форма отправляется
4.4. Иногда вылезает alert тк не все вопросы просмотрены. В случае появления он подтверждается
4.5. Задержка на 3s для полной прогрузки страницы с решением
4.6. Мыло забирает страницу
5. Перебираются карточки вопросов билета
5.1. В переменную question записывается текст вопроса
5.2. Циклом перебираются все ответы на вопрос
5.2.1. В переменную answer_text записывается текст ответам
5.2.2. Если есть отметка, что ответ верный, в начало текста ответа дописывается true
5.3. Все ответы собираются в переменную answers
5.4. В файл записывается строка вопроса в формате {number};"{question}";"{answers}"
6. Закрывается файл
7. Закрывается драйвер

Итоговый файлы имеют формат csv с разделителем ; и значения полей обрамлены "
Внутри поля ответов используется разделитель ; и добавление true для обозначения верного ответа

Номер;"Вопрос";"Ответ1;Ответ2;trueОтвет3;Ответ4"
