Парсер для сбора тестов по основам промышленной безопасности.
Собирает тесты с https://tests24.su/a1-osnovy-promyshlennoj-bezopasnosti-2019/
Написан на Phyton

Описание:
1. Подключаются библиотеки:
	selenium - для открытия в браузере решенного теста
	beautifulsoup4 - для сбора информации
2. Подключается Cromedriver. Есть настройки для headless режима, но так интереснее.
3. Перебираются 16 страниц с билетами.
4.1. На странице билета открываются все скрытые вопросы, чтобы можно было ответить на все сразу без переходов. Добавлением скрипта на JS.
4.2. В каждом вопросе selenium прожимает первый вариант.
4.3. Форма отправляется
4.4. Иногда вылезал alert тк считал, что не все вопросы просмотрены и мешал. В случае появления он подтверждается.
4.5. Задержка на 2s для полной прогрузки страницы с решением.
5.1. Создается файл для билета.
5.2. Мыло забирает страницу.
5.3. В первую строку файла записывается название билета.
5.4. Двойным циклом для каждого вопроса записываются строки с вопросом и ответами на него. Отметка "неправильно" на неверно выбранных ответах срезается заменой.
6. Закрывается файл, драйвер переходит на следующий билет (п3)
7. Закрывается драйвер.

Итоговые файлы имеют вид:
Номер билета
Вопрос 1;Ответ 1;Ответ 2;Ответ 3правильно;Ответ 4
Вопрос 2;Ответ 1правильно;Ответ 2;Ответ 3;Ответ 4
Вопрос 3;Ответ 1;Ответ 2;Ответ 3;Ответ 4правильно
....

Правильные варианты ответов помечены на сайте добавление "правильно" (взялось от галочки переведенной для читалок).
Эту особенность сайта решено было не трогать и использовать в дальнейшем по назначению.
